# jaosber.github.io

Personal site — projects, algorithms and topics that catch my attention.

**[jaosber.github.io](https://jaosber.github.io)**

## Development

```sh
npm install
npm run dev
```

## Deploy

Push to `main` — GitHub Actions builds and deploys automatically.
